# 📧 คู่มือการตั้งค่า Gmail SMTP สำหรับ Supabase Auth และ Vercel (ฟรี 100%)

คู่มือนี้จะช่วยให้คุณเชื่อมโยง **Gmail SMTP** เข้ากับทั้งระบบ **Supabase Auth** (สำหรับอีเมลยืนยันตัวตนสมัครสมาชิก) และ **Vercel** (สำหรับรหัส 2FA ใน Next.js) ทำให้ทุกอีเมลในระบบส่งฟรี และไม่มีการจำกัดสิทธิ์ผู้รับปลายทาง

---

## 🔑 ขั้นตอนที่ 1: การสร้าง Gmail App Password (รหัสผ่านแอปพลิเคชัน)
เนื่องจาก Google ปิดการใช้งาน Less Secure Apps แล้ว คุณต้องสร้างรหัสผ่าน 16 หลักแยกต่างหาก:

1. เข้าไปที่บัญชี Google ของคุณ: [Google Account Security](https://myaccount.google.com/security)
2. เปิดใช้งาน **การยืนยันตัวตนแบบสองขั้นตอน (2-Step Verification)** ในบัญชีของคุณก่อน (หากยังไม่ได้เปิด)
3. ค้นหาคำว่า **"รหัสผ่านแอป" (App Passwords)** หรือเข้าไปที่ช่องค้นหาด้านบนพิมพ์ว่า `App Passwords`
4. ตั้งชื่อแอปพลิเคชันตามต้องการ เช่น `Mangify App` แล้วกด **สร้าง (Create)**
5. คุณจะได้รหัสผ่านความปลอดภัย **16 หลักสีเหลือง** (ให้คัดลอกเก็บไว้ทันที ห้ามมีเว้นวรรค)

---

## 🛠️ ขั้นตอนที่ 2: ตั้งค่าใน Vercel (สำหรับ Next.js 2FA)
เพื่อให้อีเมลรหัส 2FA ส่งจาก Gmail ของคุณเมื่อใช้งานผ่านเว็บไซต์จริง:

1. เปิด **Vercel Dashboard** > เลือกโปรเจกต์ **Mangify**
2. ไปที่แท็บ **Settings** > **Environment Variables**
3. เพิ่มตัวแปรดังนี้:
   * **`SMTP_USER`** = `อีเมล Gmail ของคุณ` (เช่น `phuriphathem@gmail.com`)
   * **`SMTP_PASSWORD`** = `รหัสผ่านแอป 16 หลักที่สร้างจากขั้นตอนที่ 1` (ไม่มีเว้นวรรค)
4. กด **Save**
5. ทำการ Redeploy เว็บไซต์ของคุณอีกครั้งเพื่อให้ค่าใหม่ทำงาน

---

## ⚡ ขั้นตอนที่ 3: ตั้งค่าใน Supabase (สำหรับระบบสมัครสมาชิก)
ทำเพื่อปลดล็อกขีดจำกัดสมัครสมาชิก 3 ฉบับ/ชั่วโมงของ Supabase:

1. เข้าไปที่ **Supabase Dashboard** > เลือกโปรเจกต์ของคุณ
2. เมนูด้านซ้ายไปที่ **Project Settings** (ไอคอนฟันเฟืองล่างสุด) > **Auth**
3. เลื่อนลงมาที่หัวข้อ **SMTP Settings**
4. เปิดสวิตช์ **Enable Custom SMTP** เป็น **ON** (สีเขียว)
5. กรอกข้อมูลดังนี้:
   * **Sender Email**: `อีเมล Gmail ของคุณ` (ต้องเหมือนกับตัวส่ง เช่น `phuriphathem@gmail.com`)
   * **Sender Name**: `Mangify Auth`
   * **SMTP Provider**: (ปล่อยว่างไว้ หรือระบบเลือกเอง)
   * **SMTP Host**: `smtp.gmail.com`
   * **SMTP Port**: `465`
   * **SSL/TLS**: **เปิด (ON)**
   * **SMTP Username**: `อีเมล Gmail ของคุณ` (เช่น `phuriphathem@gmail.com`)
   * **SMTP Password**: `รหัสผ่านแอป 16 หลักที่สร้างจากขั้นตอนที่ 1` (ไม่มีเว้นวรรค)
6. กด **Save** ด้านล่างสุด

## ✅ ตรวจสอบผลลัพธ์
* **สมัครสมาชิกใหม่**: ลองลงทะเบียนบัญชีใหม่ ระบบจะส่งอีเมลยืนยันตัวตนออกจาก Gmail ของคุณไปยังผู้สมัครจริงโดยทันที
* **เข้าสู่ระบบ 2FA**: หากเปิดใช้งาน 2FA ไว้ เมื่อกรอกรหัสผ่านผ่านระบบจะได้รับรหัส 6 หลักทาง Gmail ทันที
